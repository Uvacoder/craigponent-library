# Fade

Fade creates a smooth UI/UX for React

## Install

React transition group is needed for Fade

```bash
yarn add react-transition-group
```

```javascript
import Fade from 'craigponent-library/dist/Fade';
```

## Basic usage

<Playground>
  <Component
    initialState={{
      in: true,
      duration: 150
    }}
  >
    {({ setState, state }) => {
      const toggle = () => setState({ in: !state.in });
      return (
        <>
          <Row>
            <Input
              label="Duration"
              value={state.duration}
              type="number"
              onChange={(e) => setState({ duration: parseInt(e.target.value, 10)})}
            />
          </Row>
          <Button size="md" onClick={toggle}>Click Me</Button>
          <Fade in={state.in} duration={state.duration}>

## Callbacks

      return (
        <>
          <p>{state.firstText}</p>
          <p>{state.text}</p>
          <Button size="md" onClick={toggle}>Click Me</Button>
          <Fade
            in={state.in}
            duration={1000}
            onEnter={() => setState({ firstText: 'Enter' })}
            onEntering={() => setState({ text: 'Entering!' })}
