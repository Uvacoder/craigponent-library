# ProgressTracker

```javascript
import ProgressTracker from 'craigponent-library/dist/ProgressTracker';
```

ProgressTracker shows an easy status update on which page the user is on. 1 - 2 - 3

## Properties

<Props of={ProgressTracker} />


## Basic Usage

<Playground>
  <Component initialState={{ activeIndex: 0 }}>
    {({ setState, state }) => (
      <>
        <ProgressTracker
          activeIndex={state.activeIndex}
          selectIndex={(i) => () => setState({ activeIndex: i })}
          items={[
            'Create Account',
            'Issuer Info',
            'Offering Info',
            'Submit',
          ]}
        />
        <br></br>
        <br></br>
        <br></br>
        <br></br>
        <Row>
          <Button
            disabled={state.activeIndex === 0}
            onClick={() => setState({ activeIndex: state.activeIndex - 1 })}
          >
            Previous Page
          </Button>
          <Button
            disabled={state.activeIndex === 3}
            onClick={() => setState({ activeIndex: state.activeIndex + 1 })}
          >
            Next Page
          </Button>
        </Row>
      </>
    )}
  </Component>
</Playground>
