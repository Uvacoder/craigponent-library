# Pagination

## Install


```bash
import Pagination from 'craigponent-library/dist/Pagination';
```


## Properties

<Props of={Pagination} />

## Basic Usage

<Playground>
  <Box style={{ background: "#fff" }} minHeight={80}>
    <Component initialState={{ activePage: 1 }}>
      {({ setState, state }) => (
        <Pagination
          activePage={state.activePage}
          setActivePage={(i) => setState({ activePage: i })}
          totalPages={20}
        />
      )}
    </Component>
  </Box>
</Playground>
