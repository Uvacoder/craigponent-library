# Dropdown

```javascript
import Dropdown from 'craigponent-library/dist/Dropdown';
```

## Properties

* Dropdown also inherits all of Tooltip props

<Props of={Dropdown} />

## Basic usage

<Playground style={{ backgroundColor: '#fff'}}>
  <Box style={{ background: "#fff", paddingTop: '20px' }} minHeight={80}>
  <Component initialState={{ value: 'ALL' }}>
    {({ setState, state }) => (
      <Row>
        <Dropdown
          onChange={(option, e) => setState({ value: option.name })}
          options={[
            { name: 'ALL', value: 'ALL' },
            { name: 'Cash Deposits', value: 'FIAT_DEPOSIT' },
            { name: 'Cash Withdrawals', value: 'FIAT_WITHDRAWAL', active: true },
            { name: 'separator-1', value: 'SEPARATOR' },
            { name: 'Token Deposits', value: 'TOKEN_DEPOSIT', disabled: true },
            { name: 'Token Withdrawals', value: 'TOKEN_WITHDRAWAL' },
          ]}
          value={state.value}
        />
      </Row>
    )}
  </Component>
  </Box>
</Playground>
