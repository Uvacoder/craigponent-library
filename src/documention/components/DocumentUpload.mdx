# DocumentUpload

DocumentUpload facilitates uploading some form of media. Supports Dragging n dropping

## Install

react-dropzone is needed for DocumentUpload

```bash
yarn add react-dropzone
```

```bash
import DocumentUpload from 'craigponent-library/dist/DocumentUpload';
```
