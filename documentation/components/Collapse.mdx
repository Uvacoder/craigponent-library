# Collapse

Collapse creates a smooth transition to hide and show content

## Install

React transition group is needed for the collapse

```bash
yarn add react-transition-group
```

```javascript
import Collapse from 'craig-component-library/dist/Collapse'
```
or
```javascript
import UncontrolledCollapse from 'craig-component-library/dist/UncontrolledCollapse'
```
